# webpack-5-template

***
### Install:

1. Fork this repository
2. Open project in IDE
3. Switch branch - webpack-template
4. Run in console: <br>
    `npm install --save-dev` <br>
    `npm prune --save-dev` <br>
    `npm audit fix`
